## ğŸš€ Installation (Development Mode)

Since this theme is in development, you can install it manually:

1.  **Download/Clone the repository:**
    ```bash
    git clone https://github.com/rifatcholakov/Cozy-Spring-Meadow.git
    ```
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **"Developer mode"** (toggle in the top right).
4.  Click **"Load unpacked"** and select the `Cozy-Spring-Meadow` folder.
